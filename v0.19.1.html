Release v0.19.1

Published: Tue 04 August 2015
By Amjith

In Blog.

tags: python release changelog postgres

Pgcli is a command line interface for Postgres database that does auto-completion and syntax highlighting. You can install this version by:

$ pip install -U pgcli

Check detailed instructions if you're having difficulty.

This is a minor bug fix release to fix a crashing bug during autocompletion.

Bug Fixes:

  • Fix an autocompletion bug that was crashing the completion engine when unknown keyword is entered. (Thanks: Darik Gamble)
