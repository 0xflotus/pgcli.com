Thanks

pgcli was made possible by the existence of these wonderful people and libraries.

Libraries:

Python Prompt Toolkit - An awesome library for building interactive command line interfaces.

Click - Basic building blocks for a CLI.

Tabulate - Pretty printing tabular data.

Sqlparse - Sql Parsing library.

Psycopg - A solid library for talking to postgres.

Pygments - Syntax highlighting.
